# Belco Leave Portal — Frontend

Angular 20 SPA for the Belco Leave Portal. See the [root README](../README.md) for full project documentation.

## Development

```bash
npm install
ng serve
```

App runs on `http://localhost:4200`.

## Build

```bash
ng build
```

Output is written to `dist/leave-platform-ui/browser/`.

## Deployment

Deployed to Vercel. The `vercel.json` at this directory's root configures the build command and output directory.
